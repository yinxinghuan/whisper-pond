# Requirements

## 1. Overview

《心事水面》是一款 60 秒感官互动玩具，玩家输入一句心事后通过触摸水面投下字石、制造涟漪、积累回声值并在结束时得到一张安静的结算卡。

## 2. Visual Design

- 画面为固定竖屏体验，逻辑画布尺寸 390px × 680px，外层容器圆角 24px，背景色从顶部 `#07111f` 渐变到底部 `#182936`。
- 主水面覆盖全屏，使用深蓝 `#0b1f33`、青绿 `#3ad8c6`、月白 `#f4f7ff` 三组颜色绘制；每个水波外圈线宽 1.5px，最大半径 180px，透明度从 0.72 衰减到 0。
- 顶部 HUD 位于安全区下方 18px，左侧显示剩余时间，字号 12px、字距 0.22em、颜色 `rgba(238,248,255,0.78)`；右侧显示回声值，字号 28px、字体重量 700、颜色 `#f4f7ff`。
- 中央输入卡宽度 310px，高度 178px，背景 `rgba(5,12,22,0.62)`，边框 `1px solid rgba(205,242,255,0.24)`，模糊 18px，圆角 18px。
- 输入框高度 64px，字号 18px，颜色 `#f8fbff`，占位文字颜色 `rgba(248,251,255,0.44)`；开始按钮高度 46px，圆角 999px，背景 `#d7fff5`，文字 `#09222a`。
- 字石以短词或单字显示，最多 9 枚同时存在；每枚字石字号 13px，颜色 `rgba(248,251,255,0.82)`，带 `0 0 12px rgba(90,235,220,0.35)` 光晕，并以 4.8s 漂浮淡出。
- 空闲演示在未开始时每 1.7s 自动生成 1 个小涟漪，最多保留 5 个；开始后不再自动堆积。
- 结算卡宽度 318px，高度 244px，展示最终回声值、历史最高、最后一句心事和 3 个操作按钮；卡片不嵌套其它卡片。
- 素材清单：`public/img/aigram.svg` 作为右下角 52px 水印；其余视觉全部由 CSS 和 Canvas 生成。

## 3. Game Mechanics

- 初始时间为 60 秒，开始后每帧根据 `performance.now()` 更新倒计时，倒计时到 0 进入结算。
- 玩家提交的心事按空格、标点和字符拆成词粒；中文连续字符串按 1-2 字片段循环，英文按单词循环。
- 每次有效触摸生成 3 层水波：半径起始 8px、14px、20px，扩张速度分别为 140px/s、110px/s、82px/s，生命周期分别为 1.4s、1.9s、2.3s。
- 每次有效触摸生成 1 枚字石，字石初始位置为触摸点，漂移速度 x 为 -14px/s 到 14px/s，y 为 -28px/s 到 -8px/s，生命周期 4.8s。
- 连续 1.8 秒内触摸会累积 combo，combo 从 1 到 5；单次得分为 `10 + combo * 3`，combo 超过 5 时仍按 5 计算。
- 水面粒子光点最多 80 个，触摸时新增 8 个，生命周期 1.2s 到 2.2s；超过上限时移除最旧粒子。
- 最高分使用 `localStorage` 键 `whisper_pond_best` 持久化。
- 无死亡、无惩罚；miss 不存在，任意水面触摸都算有效互动。

## 4. Controls

- 开始页：点击输入框可编辑心事；点击开始按钮触发音频解锁并进入游戏。
- 游戏中：使用 `pointerdown` 在水面任意位置投下字石；不绑定 `pointermove`，避免移动端 WebView 高频事件卡顿。
- 键盘：输入框中按 Enter 且内容非空时开始；游戏结束页按钮可通过点击触发。
- 触摸：所有按钮和水面交互均使用 `pointerdown`，交互元素设置 `touch-action: none`、`user-select: none` 和 `-webkit-tap-highlight-color: transparent`。

## 5. Win / Lose Conditions

- 胜利条件：60 秒倒计时结束即完成一局，没有失败状态。
- 结算界面显示最终回声值、历史最高、最高 combo、投下的字石数量、最后一句心事。
- 结算界面提供 3 个操作：再来一次、换一句心事、返回首页；再来一次保留当前心事，换一句回到输入态。
- 开始页、游戏中、结束页为三个互斥状态；开始前和结束后水面仍保持轻微可视运动，但不计分。

## 6. Sound Effects

- 开始：两枚 sine 波，440Hz 到 660Hz 持续 0.12s，随后 660Hz 到 880Hz 持续 0.16s，音量 0.055。
- 投下字石：triangle 波 190Hz 到 130Hz 持续 0.11s，音量 0.045，同时叠加 sine 波 520Hz 持续 0.18s，音量 0.018。
- combo ≥ 2：每次触摸附加 sine 波，频率 `620 + combo * 40` Hz，持续 0.09s，音量 0.022。
- 结算：四音上升 pad，频率 392Hz、523Hz、659Hz、784Hz，每音持续 0.22s，间隔 0.07s，音量 0.04。
- 按钮点击：square 波 520Hz 到 360Hz 持续 0.045s，音量 0.025。
